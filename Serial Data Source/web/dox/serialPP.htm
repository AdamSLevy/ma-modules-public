Configuration

The data type governs how to interpret the data. Current parsing is done with the Integer, Double, and Boolean parse functions. The point identifier is used if a terminator is being used to match the message index group as matched by the message regex, which determines if the value regex should be applied. If a match is found, the group at value index is sent to the parser.

If no terminator is being used, the value regex is applied judiciously and, upon finding a match, the group at value index is sent to parsing.
