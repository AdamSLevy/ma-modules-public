Overview

This data source reads a file as an ASCII character stream on a line by line basis.

Configuration

Every data source requires a Name, which can be any description, and an XID, which must be unique among all data sources. The polling configuration specifies how frequently the data source checks the file. The File Path points to the file you want to access information from. The file path is absolute, but networked paths are not supported. Use the "Check" button to verify your file path references an existing file.

Polling Configuration

Quantize will cause the data source startup to be delayed so that pollings occur at a "rounded" point in time (e.g. if polling is every 10 seconds, polls will occur on the 10th second, the 20th second, etc). Otherwise, polling will begin immediately. While using quantized polls Cron pattern is not an option.

Update periods define the period at which the poll will occur. For example, if you choose "2 minutes", then every two minutes the algorithms that change the virtual point values for that data source will be run.

Cron pattern is optional but when set will disable the Update period setting. Cron patterns can be used to schedule the data source to run at flexible intervals.
